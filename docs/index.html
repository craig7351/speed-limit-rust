強大的核心與優雅的介面

專為需要精確控制網路流量的進階使用者設計，效能與美觀兼具

全域 & 程序限速

可設定整體網路頻寬上限，也可針對個別程式（如 Chrome、Steam）設定獨立的下載/上傳速度限制。

Token Bucket 演算法

核心採用精確的流量整形技術，最大突發量為 1 秒流量，單一封包最多延遲 100ms 避免 TCP 連線超時。

即時速度監控

每 500ms 更新下行/上行速度，並即時顯示前 5 名流量最高的程式，網路使用狀況一目了然。

macOS 風格 GUI

使用 egui 打造流暢的圖形化介面，淺色主題、卡片式佈局與圓角設計，提供極佳的使用者體驗。

安全停止機制

使用 WinDivert 核心 API 確保停止限速後，正確移除核心過濾器，網路立即無縫恢復正常。

智慧程序偵測

結合 Flow 層即時連線追蹤與 Windows 系統掃描，自動列出所有可限速程式，並支援設定檔自動儲存。

Installation Guide

# 1. 下載最新版本

前往 GitHub Releases 下載 SpeedLimit-Windows-x64.zip 並解壓縮到任意資料夾

# 2. 執行程式 (自動請求管理員權限)

.\speed-limit.exe

重要注意事項：

請確保 speed-limit.exe、WinDivert.dll 與 WinDivert64.sys
這三個檔案**必須放在同一個資料夾下**才能正常啟動。
